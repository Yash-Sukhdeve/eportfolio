# images/

Put your photos and project screenshots in this folder.

| File name | What it is |
|-----------|------------|
| `profile.jpg` | Your profile photo — used in `index.html` |
| `project1.png` | Screenshot of your first project — used in `projects.html` |
| `project2.png` | Screenshot of your second project — used in `projects.html` |

## Rules for file names

- Use **lowercase letters only** — `profile.jpg` not `Profile.jpg`
- Use **no spaces** — `my-project.png` not `my project.png`
- If you use a different name, update the `src="images/..."` in the HTML to match

## How to upload on GitHub

1. In your repo, click **Add file → Upload files**
2. Drag your image in (or click to browse)
3. Scroll down and click **Commit changes**

The image will now be at `images/your-file-name.jpg` — use that exact path in the HTML.
